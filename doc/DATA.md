# Данные и назначение репозитория

Документ описывает, **какие данные** использует RAG, **откуда** они взяты и **что именно** попадает в индекс.

---

## Назначение репозитория

**Для кого:** студенты курса «Корпоративные данные» и все, кто учится строить RAG с нуля.

**Что демонстрирует:**

- полный offline-pipeline: сырые новости → документы → чанки → TF-IDF индекс → поиск → ответ;
- ответ **только по найденным фрагментам** с указанием источника (`doc_id`, score, заголовок);
- явный **отказ**, если релевантного контекста нет (защита от галлюцинаций);
- Streamlit UI для интерактивной проверки с регулировкой порога и подсветкой слов запроса.

**Границы MVP:**

- поиск по **словам** (TF-IDF + cosine), не по эмбеддингам и не через LLM;
- demo-режим генерации без внешних API и ключей;
- локальный корпус: 1500 новостных статей, ~11 000 чанков;
- это **учебный шаблон**, а не production-система.

## Источник данных

| Источник | Откуда | Комментарий |
|----------|--------|-------------|
| [`IlyaGusev/gazeta`](https://huggingface.co/datasets/IlyaGusev/gazeta) | HuggingFace Hub | Корпус новостей сайта **Gazeta.ru** (текст статьи, заголовок, дата, url, аннотация). Скачивается публично, без авторизации. |
| Подготовленный корпус | `data/raw/datasets.json` | 1500 записей: `id`, `name`, `text`, `section`, `date`, `url`. **Коммитится** — готовый набор. |
| Скрипт подготовки | `scripts/prepare_datasets.py` | Стримит датасет, фильтрует деловые рубрики, чистит текст, пишет `datasets.json`. |

**Лицензия:** датасет `IlyaGusev/gazeta` распространяется на условиях, указанных на странице датасета (исследовательское/учебное использование). Тексты принадлежат Gazeta.ru. В репозиторий включена ограниченная учебная выборка.

**Дата подготовки выборки:** июнь 2026. Сами статьи относятся к более раннему периоду (новости Gazeta.ru, преимущественно 2010–2013 гг.).

## Как формируется корпус

`prepare_datasets.py` оставляет только **деловые и экономические рубрики**, определяя рубрику по разделу в `url` (`https://www.gazeta.ru/<section>/...`):

| Рубрика (`section`) | Тематика | Статей в выборке |
|---------------------|----------|:----------------:|
| `business`   | бизнес, компании, рынки | 703 |
| `financial`  | финансы, банки, валюта, инфляция | 619 |
| `techzone`   | технологии, связь, гаджеты | 96 |
| `realty`     | недвижимость, жильё | 82 |
| **Итого**    | | **1500** |

Параметры отбора (в `prepare_datasets.py`): `MAX_ARTICLES=1500`, `MIN_TEXT_LEN=400` символов, дедупликация по заголовку.

## Что индексируем

| Поле / артефакт | Индексируется? | Где используется |
|-----------------|:--------------:|------------------|
| `text` (тело статьи) | **Да** | TF-IDF матрица, поиск, ответ |
| `name` (`[section] заголовок`) | Нет (метаданные) | UI и источники — подпись документа |
| `doc_id` | Нет (метаданные) | UI и источники — идентификатор статьи |
| `section`, `date`, `url` | Нет (метаданные) | трассировка происхождения в `documents.jsonl` |

**Pipeline:**

```
datasets.json → documents.jsonl → chunks.jsonl → vectorizer.pkl + matrix.npz
```

- **Чанки:** нарезка по абзацам, max 600 символов, overlap 100 (`app/chunker.py`).
- **Индекс:** TF-IDF с биграммами (`ngram_range=(1,2)`), русскими стоп-словами и `min_df=2` (`scripts/build_index.py`).
- **Поиск:** cosine similarity по TF-IDF векторам, top-k (`app/retriever.py`).

---

## Что НЕ индексируем

| Не индексируется | Причина |
|------------------|---------|
| Аннотация (`summary`) исходного датасета | В RAG идёт полный текст статьи, а не готовое резюме |
| Метаданные (`section`, `date`, `url`) | Используются как источники/трассировка, не как поисковый текст |
| Кеш HuggingFace | Только для пересборки `datasets.json`, в `.gitignore` |
| `data/processed/*`, `data/index/*` | Промежуточные артефакты, пересобираются скриптами |
| Секреты, API-ключи | Demo-режим без внешних LLM |

## Демонстрация (3 ответа + 1 отказ)

| Вопрос | Ожидание |
|--------|----------|
| **инфляция и рост тарифов** | ответ, `doc_id=0` («Тарифы инфляцию не остановят»), score ≈ 0.38 |
| **курс рубля и доллар** | ответ, `doc_id=335` («Доллар обмякнет»), score ≈ 0.38 |
| **Центробанк ставка рефинансирования** | ответ, `doc_id=1` («Греф задирает ставку»), score ≈ 0.33 |
| **Как приготовить борщ?** | **отказ** — таких тем в корпусе нет (score = 0.00) |

## Как обновить данные

```bash
uv sync --extra data                              # доустановить datasets
uv run python scripts/prepare_datasets.py         # пересобрать datasets.json
uv run python scripts/build_index.py              # пересобрать индекс
```
