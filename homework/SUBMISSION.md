## Комментарий

**Что реализовано:** учебный RAG по деловым и экономическим новостям Gazeta.ru.
Полный pipeline: подготовка данных → ingestion → chunking → TF-IDF индекс →
retrieval (cosine top-k) → demo-ответ с источниками → Streamlit UI. Есть явный
отказ при отсутствии релевантного контекста.

**Данные:** датасет `IlyaGusev/gazeta` (HuggingFace), деловые рубрики
(business / financial / realty / techzone). Корпус — **1500 статей**, после
нарезки — **~11 154 чанка**.

**Демонстрация:** 3 вопроса с ответами и источниками (инфляция и тарифы; курс
рубля и доллар; ставка ЦБ) + 1 отказ («Как приготовить борщ?», score = 0.00).
Тесты: **11 шт., все green** (`pytest`).

**Улучшения (реализовано):**
1. Streamlit UI — регулируемые top-k и порог релевантности, подсветка слов запроса.
2. Качество поиска — русские стоп-слова + биграммы в TF-IDF (чинит отказ на
   нерелевантные запросы).
Направления развития (эмбеддинги, векторная БД, LLM, reranking, eval) — в `IMPROVEMENTS.md`.
